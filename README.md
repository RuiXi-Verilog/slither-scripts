# Slither scripts

- In `erc20` folder, a script to detect ERC20 interfaces
- In `modifier` folder, a script to detect functions with a given modifier

## Requirements
- [Python 3](https://www.python.org/downloads/)

## Install
1. `pip3 install -r requirements.txt`
